Download Elementary OS Gratis

2020-04-06 [Updated: 2020-04-06] :: @Kripul
#Linux  #ElementaryOS 

Cara Download Elementary OS Hera Gratis⌗

Salah satu distro dengan tampilan yang sangat-sangat menarik adalah Elementary OS, distro keturunan dari Ubuntu ini menggunakan DE (Desktop Environment) sendiri yang bernama Pantheon.

Banyak yang masih bingung bagaimana cara mendownload distro ala-ala MacOS ini. karena di halaman resminya seperti harus membayar untuk mendownload file ISO nya.

Berikut ini cara Download Elementary OS Hera Gratis⌗

Pertama : Buka halaman resmi Elementary OS di https://elementary.io/id/

Kedua : Kemudian di kolom “Sesuaikan” isi dengan angka 0 lalu klik tombol “Unduh Elementary OS”

Ketiga : Kalau muncul pop up seperti digambar, klik Unduh untuk mendownload lewat browser atau bisa juga mendownload menggunakan torrent dengan mengklik ikon magnet disamping tulisan Unduh

Proses download akan segera dimulai, tinggal tunggu prosesnya sampai selesai.

Terimakasih!
