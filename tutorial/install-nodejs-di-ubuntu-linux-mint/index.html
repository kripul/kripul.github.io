Install NodeJs di Ubuntu / Linux Mint

2019-01-11 :: Nixpoin

Install NodeJs di Ubuntu / Linux Mint⌗

Ada dua cara untuk menginstall node.js di Ubuntu / Linux Mint, yaitu menggunakan repositori dari pihak Ubuntu atau menggunakan repositori NodeSource

Perbedaannya adalah, kita bisa memilih versi node.js jika kita menggunakan repositori dari NodeSource, untuk sekarang versi LTS dari Node.Js adalah versi 10 tetapi kalau kita menggunakan repositori resmi dari Ubuntu 18.04 masih menggunakan Node.Js versi 8

Install Node.Js menggunakan repositori Ubuntu⌗

update dahulu repositori kita dengan

<pre class="wp-block-preformatted">sudo apt-get update

Lalu Install Node.Js dengan perintah

<pre class="wp-block-preformatted">sudo apt-get install nodejs

Sekarang kita cek versi yang terinstall

<pre class="wp-block-preformatted">nodejs -v

Hasilnya seperti gambar di bawah

Sebelumnya, kita uninstall dulu Node.Js yang sudah kita install barusan dengan perintah

<pre class="wp-block-preformatted">sudo apt-get remove nodejs

Selanjutnya kita download file sh dari NodeSource, file sh ini akan menambahkan PPA dari NodeSource ke repositori kita

curl -sL https://deb.nodesource.com/setup_10.x -o nodesource_setup.sh

Jalankan file nodesource_setup.sh dengan perintah

<pre class="wp-block-preformatted">bash nodesource_setup.sh

PPA dari NodeSource sudah ditambahkan, sekarang kita install Node.Js dengan perintah

<pre class="wp-block-preformatted">sudo apt-get install nodejs

Sekarang kita coba cek versi Node.js yang tersintall
