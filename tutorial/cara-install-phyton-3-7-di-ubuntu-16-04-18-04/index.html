Cara Install Phyton 3.7 di Ubuntu 16.04 / 18.04

2019-01-14 :: Nixpoin

Apa Itu Phyton?⌗

Phyton adalah salah satu bahasa pemograman yang populer di dunia, karena simple dan sangat mudah dipelajari bagi pemula. Kita bisa membuat game, website, aplikasi desktop dengan Phyton. Dan Phyton dapat berjalan di Linux, Windows ataupun MacOs

Phyton 3.7 adalah versi Phyton mayor rilis terbaru yang memuat beberapa pembaharuan, kalian bisa melihat pembaharuan lengkapnya disini https://docs.python.org/3/whatsnew/3.7.html

Tutorial kali ini kita akan mencoba menginstall Phyton di sistem Ubuntu 18.04 namun bisa juga diterapkan di Ubuntu 16.04 atau pun Linux Mint bahkan ElementaryOS.

Cara Install Phyton 3.7⌗

Update Install Paket Penunjang

<pre class="wp-block-preformatted">sudo apt update<br></br>sudo apt install software-properties-common

Selanjutnya kita tambahkan PPA dari deadsnakes

<pre class="wp-block-preformatted">sudo add-apt-repository ppa:deadsnakes/ppa

Lalu tekan tombol ENTER untuk melanjutkan

<pre class="wp-block-preformatted">Press [ENTER] to continue or Ctrl-c to cancel adding it.

PPA dari deadsnakes sudah ditambahkan di repo list kita, update dahulu daftar paket

<pre class="wp-block-preformatted">sudo apt update

Sekarang kita install phyton 3.7 dengan perintah

<pre class="wp-block-preformatted">sudo apt install python3.7

Sekarang kita cek apakah Python sudah terinstall dengan benar

<pre class="wp-block-preformatted">python3.7 --version

Maka hasilnya seperti

<pre class="wp-block-preformatted">Python 3.7.2

Selamat mencoba!
