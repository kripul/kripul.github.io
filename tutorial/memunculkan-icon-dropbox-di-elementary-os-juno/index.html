Memunculkan Icon Dropbox di Elementary OS Juno

2019-01-20 :: Nixpoin

Memunculkan Icon Dropbox di Elementary OS Juno⌗

Salah satu masalah yang muncul di Elementary OS Juno adalah, hilangnya dukungan wingpanel indicator sehingga icon Dropbox tidak muncul.

Install wingpanel-indicator-ayatana⌗

Download file deb wingpanel-indicator-ayatana

<pre class="wp-block-preformatted">wget http://ppa.launchpad.net/elementary-os/stable/ubuntu/pool/main/w/wingpanel-indicator-ayatana/wingpanel-indicator-ayatana_2.0.3+r27+pkg17~ubuntu0.4.1.1_amd64.deb

Install file wingpanel-indicator-ayatana

<pre class="wp-block-preformatted">sudo dpkg -i wingpanel-indicator-ayatan*.deb

Rubah file indicator application desktop⌗

Selanjutnya kita edit file /etc/xdg/autostart/indicator-application.desktop

<pre class="wp-block-preformatted">sudo nano /etc/xdg/autostart/indicator-application.desktop

Cari baris

<pre class="wp-block-preformatted">OnlyShowIn=Unity;GNOME;

dan ubah menjadi

<pre class="wp-block-preformatted">OnlyShowIn=Unity;GNOME;Phanteon;

Proses terakhir, silahkan log out atau restart OS kalian dan icon indicator yang sebelumnya tidak muncul di Elementary OS Juno sudah muncul.
