Cara Install Sublime Text 3 di Ubuntu 18.04 atau Linux Mint

2019-01-25 :: Nixpoin

Pendahuluan⌗

Sublime Text adalah salah satu teks editor favorit saya, Sublime sangat powerfull dan banyak sekali plugin-plugin yang bermanfaat. Sublime sendiri dapat berjalan di sistem berbasin Windows, MacOS atau pun Linux.

Kali ini kita akan mencoba untuk menginstall Sublime Text 3 di Ubuntu atau pun Linux Mint.

Untuk Linux Mint sendiri sudah menyediakan Sublime Text di repositori mereka tapi biasanya paket tersebut bukan paket terbaru. Maka kali ini kita akan menginstall Sublime Text menggunakan repositori dari PPA pihak Sublime Text

Tambahkan Repositori Sublime3⌗

Jalankan perintah di bawah ini

<pre class="wp-block-preformatted">echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list

Selanjutnya kita download GPG key dari repositori Sublime3 dan menambahkannya ke apt key dengan perintah

<pre class="wp-block-preformatted">wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

Install Sublime Text 3⌗

Setelah menambahkan repositori, kita harus melakukan update repositori kita dulu

<pre class="wp-block-preformatted">sudo apt-get update

Proses selanjutnya adalah instalasi Sublime Text, kita bisa menginstall Sublime Text 3 dengan perintah

<pre class="wp-block-preformatted">sudo apt-get install sublime-text

Tunggu sampai selesai.
