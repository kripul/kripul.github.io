Cara Install zsh dan oh-my-zsh di Centos, Ubuntu, Linux Mint

2019-01-09 :: Nixpoin

Zsh adalah Unix Shell yang dapat digunakan sebagai login sheel, beberapa perbedaannya dengan bash adalah

  • Validasi perintah : cukup ketikkan saja “ap” dan tekan tombol tab nanti akan muncul perintah dengan awalan “ap”
  • Koreksi kesalahan pengetikan kapital : misal kita mengetikkan download, lalu kita tekan tab maka akan berubah menjadi Downloads
  • Plugin dan tema : kita bisa menambahkan plugin seperti plugin “Git” dan juga mempercantik zsh dengan oh-my-zsh

Lalu, bagaimana cara menginstallnya?

Step pertama : Install git, zsh dan curl⌗

untuk debian, ubuntu, linux mint, gunakan perintah berikut

<pre class="wp-block-preformatted">sudo apt install zsh git curl

untuk Centos, gunakan perintah

<pre class="wp-block-preformatted">sudo yum install zsh git curl

setelah itu kita cek apa benar zsh sudah terinstall di sistem kita dengan perintah

<pre class="wp-block-preformatted">zsh --version

Selanjutnya kita akan mempercantik tampilan dari zsh dengan oh-my-zsh
jalankan perintah di bawah ini untuk menginstall oh-my-zsh, nanti akan diminta untuk memasukkan password root, untuk mengubah dengan shell dari bash ke zsh

<pre class="wp-block-preformatted">sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

Kalian bisa mencari banyak sekali tema oh-my-zsh di link berikut

https://github.com/robbyrussell/oh-my-zsh/wiki/themes

Untuk mengubah temanya, edit file ~/.zshrc

<pre class="wp-block-preformatted"> nano ~/.zshrc

Ganti nilai dari ZSH_THEME sesuai dengan selera kalian. Tekan CTRL + x untuk menyimpan hasil perubahan.

Dan silahkan Login lagi untuk melihat perubahan di terminal kalian.

Selamat mencoba!
