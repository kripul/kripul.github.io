Cara Install Gradle di Sistem Operasi Centos 7

2019-01-17 :: Nixpoin

Apa itu Gradle⌗

Gradle adalah suatu fitur build automation yang open source yang biasanya digunakan untuk proyek berbasis bahasa pemograman java sebagaimana Maven atau Ant.

Tutorial kali ini akan membahas cara install Gradle di sistem operasi Centos 7, berikut ini caranya

Install OpenJDK⌗

Gradle membutuhkan Java JDK / JRE versi 7 ke atas, disini kita akan menggunakan OpenJDK versi 8

<pre class="wp-block-preformatted">sudo yum install java-1.8.0-openjdk-devel

Untuk mengecek apakah OpenJDK sudah terinstall, gunakan perintah di bawah ini

<pre class="wp-block-preformatted">java -version

Download Gradle dari situs resminya https://gradle.org/releases/, pada tulisan ini dibuat versi terbarunya adalah 5.1.1

<pre class="wp-block-preformatted">wget https://downloads.gradle.org/distributions/gradle-5.1.1-bin.zip

Selanjutnya kita extract file Gradle yang kita download tadi ke folder /opt/gradle

<pre class="wp-block-preformatted">sudo unzip -d /opt/gradle gradle-5.1.1-bin.zip

Setting environment variable untuk Gradle⌗

Buat sebuah file gradle.sh di folder /etc/profile.d/ dengan perintah

<pre class="wp-block-preformatted">sudo nano /etc/profile.d/gradle.sh

Lalu pastekan baris berikut

<pre class="wp-block-preformatted">export GRADLE_HOME=/opt/gradle/gradle-5.1.1<br></br>export PATH=${GRADLE_HOME}/bin:${PATH}

Selanjutnya kita jadikan file gradle.sh tersebut menjadi executable

<pre class="wp-block-preformatted">sudo chmod +x /etc/profile.d/gradle.sh

Lanjut kita akan meng-load file tersebut dengan perintah

<pre class="wp-block-preformatted">source /etc/profile.d/gradle.sh

Proses instal Gradle sudah selesai, seperti biasa kita perlu cek terlebih dahulu apakah sudah berhasil atau belum proses instalnya.
Jalankan perintah berikut

<pre class="wp-block-preformatted">gradle -v
