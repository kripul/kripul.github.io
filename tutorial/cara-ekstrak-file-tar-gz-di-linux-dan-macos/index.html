Cara Ekstrak File Tar Gz di Linux dan MacOS

2017-04-21 [Updated: 2017-04-21] :: @Kripul
#bash  #CLI 

Jika kalian adalah termasuk orang sedang belajar dengan Open Source, sudah pasti kalian akan sering menjumpai file berektensi .tar.gz. Paket program atau aplikasi open source umumnya tersedia dalam format .tar.gz dan .zip.

Di sistem operasi berbasis Unix seperti Linux dan MacOS sudah terinstall perintah tar kependekan dari “Tape ARchive” yang fungsinya untuk membuat dan mengekstrak arsip tar dan perintah tar ini mendukung tipe kompresi yang sangat banyak seperti gzip, bzip2, lzip, lzma, lzop dan xz

Tipe kompresi yang paling populer dan biasa kita temui adalah gzip, biasanya file tersebut berektensi ,tar.gz atau .tgz. Simplenya, file .tar.gz atau .tgz adalah file arsip tar yang menggunakan algoritma kompresi gzip

Di tutorial kali ini kita akan belajar cara mengekstrak file arsip tar

Ekstrak File tar.gz⌗

Untuk mengekstrak file tar.gz kita bisa menggunakan opsi --extract atau -x , kemudia opsi f untuk menentukan nama file yang ingin kita ekstrak

tar -xf namafile.tar.gz

tar -xf namafile.tar.bz2

Perintah tar secara otomatis mendeteksi algoritma kompresi file tersebut.

Kita bisa menambahkan opsi v untuk mode verbose sehingga menampilkan file apa saja yang diekstrak

tar -xvf namafile.tar.gz

Menentukan Lokasi Hasil Ekstrak File Tar⌗

Tambahkan opsi -C untuk menspesifikkan lokasi dari hasil ekstrak

tar -xvf namafile.tar.gz -C /home/nixpoin/files

Melihat Isi File tar⌗

Sebelum mengekstrak, ada baiknya kita mengintip apa saja isi file tar yang ingin kita ekstrak, caranya

tar -tvf namafile.tar.gz

Ekstrak satu file⌗

Kita juga bisa mengekstrak beberapa file saja yang kita inginkan,

tar -xvf namafile.tar.gz namafiledidalamfiletar.jpg

Mudah bukan? Untuk belajar perintah tar lebih lanjut silahkan baca di https://linux.die.net/man/1/tar
