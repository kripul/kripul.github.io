Cara Install PPTP VPN Ubuntu 18.04 server

2019-12-17 :: Nixpoin

PPTP atau The Point-to-Point Tunneling Protocol adalah salah satu jenis VPN yang sering digunakan untuk point to point / menghubungkan dua jaringan network yang berbeda.
Misalnya kita ingin menghubungkan jaringan lokal di kantor A agar bisa diakses oleh kantor B atau sebaliknya, sama seperti OpenVPN, semua transmisi data juga terenkripsi sehingga semua transmisi data aman. Bedanya, jika OpenVPN bisa berjalan dengan jaringan UDP dan TCP maka PPTP hanya bisa menggunakan jaringan TCP saja.

Install paket PPTP⌗

Lakukan update repositori lebih dahulu

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo apt update

Lalu install paket PPTP

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo apt install pptpd -y

Menambahkan DNS⌗

Paket PPTP sudah terinstall sekarang kita tinggal menkonfigurasinya, yang pertama kita ubah adalah setingan DNS yang akan digunakan oleh si client dari PPTP VPN
edit file /etc/ppp/pptpd-options

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo nano /etc/ppp/pptpd-options

Lalu tambahkan baris

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">ms-dns 8.8.8.8
ms-dns 8.8.4.4

Kalian bisa juga menggunakan DNS dari Cloudflare atau DNS lainnya.

Menambahkan user VPN PPTP⌗

Edit file /etc/ppp/chap-secrets

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo nano /etc/ppp/chap-secrets

Untuk menambahkan user, tambahkan baris baru dengan format

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">namauser1    pptpd   passworduser1   *
namauser2    pptpd   passworduser2   *

Konfigurasi Alokasi Private IP⌗

User sudah berhasil ditambahkan, sekarang kita konfigurasi private IP yang akan didapatkan oleh client VPN PPTP
Ubah file /etc/pptpd.conf

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo nano /etc/pptpd.conf

Lalu tambahkan bari berikut

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">localip 10.0.0.1
remoteip 10.0.0.100-200

Aktifkan IP Forwading⌗

Fungsi dari IP Forwading ini adalah agar client bisa melakukan koneksi ke internet melalui PPTP VPN server.

Ubah file /etc/sysctl.conf\

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo nano /etc/sysctl.conf

Lalu tambahkan baris berikut

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">net.ipv4.ip_forward = 1

Sekarang kita aktifkan konfigurasi barunya dengan perintah

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo sysctl -p

Selanjutnya tambahkan firewall iptables agar client bisa melakukan koneksi ke internet melalui interface netwrok server kita.

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE

Disini interface jaringan saya menggunakan eth0, silahkan cek interface network kalian dengan perintah

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">ip address

Jalankan PPTP VPN⌗

Oke, ini proses terakhir yaitu mengaktifkan PPTP nya dengan perintah

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo systemctl start pptpd

Lalu kita tambahkan agar proses PPTP nya autostart dengan perintah

<pre class="EnlighterJSRAW" data-enlighter-group="" data-enlighter-highlight="" data-enlighter-language="generic" data-enlighter-linenumbers="" data-enlighter-lineoffset="" data-enlighter-theme="" data-enlighter-title="">sudo systemctl enable pptpd

Proses sudah selesai, sekarang kita bisa menggunakan VPN PPTP di device kita.
