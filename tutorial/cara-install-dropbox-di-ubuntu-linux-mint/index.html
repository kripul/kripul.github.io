Cara Install Dropbox Di Ubuntu / Linux Mint⌗

Dropbox adalah salah satu penyedia file sharing untuk berbagi data ataupun file berkas, tersedia layanan berbasis Free dan Premium.

Lalu bagaimana cara untuk menginstallnya di Ubuntu atau Linux mint?

Step : Pertama download dropbox⌗

Download dropbox menggunakan browser kalian dari https://www.dropbox.com/install-linux

Step Kedua : Install dropbox⌗

Buka terminal lalau pindah ke folder Downloads / lokasi file dropbox yang kita download tadi

cd Downloads

Lalu install

<pre class="wp-block-preformatted">sudo dpkg -i dropbox*.deb

nanti akan ada pesan error

Untuk mengatasi pesan error di atas, jalankan perintah

Proses instalasi sudah selesai dan dropbox sudah ada di menu / dash
